Detailed Description

Represents an aviary (enclosure) in the zoo.

Each Aviary contains animals, has an assigned employee, and stores physical attributes such as type, area, and capacity. It is implemented as a graph vertex.

The Aviary class inherits from Vertex and extends it with additional zoo-related data such as resident animals and assigned caretaker.
