Detailed Description

Represents a user account in the Zoo Management System.

The Account class stores user authentication details (username, password hash) and authorization role. It provides convenient serialization for saving to persistent storage and deserialization for loading from files or databases.

Passwords are not stored in plaintext — instead, a secure hash is kept. The class supports conversions between Role enums, integer codes, and their string representations for storage or UI display.
