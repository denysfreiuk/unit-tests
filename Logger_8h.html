Detailed Description

Declaration of the Logger class used for system logging.

Author
Denys Freyuk
Date
27.10.2025
Version
1.1

The Logger class provides a simple but effective logging utility for the Zoo Management System. It supports multiple log levels, colored console output, and optional file logging for debugging and system monitoring.

The logger supports four log levels (DEBUG, INFO, WARN, ERROR), which can be enabled or disabled individually. Messages are timestamped and color-coded in the console.
