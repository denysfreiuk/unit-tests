Detailed Description

Handles the main user interface routing by user role.

The Menu class determines which role-specific interface to display (Administrator, Manager, or Employee) and delegates control to the corresponding menu namespace. It acts as a high-level controller in the application’s user interaction layer.

Note
This class does not maintain any state — all methods are static.

Displays the appropriate menu interface for a given user.

Parameters
usernameThe username of the currently logged-in account.
roleThe user’s role (Admin, Manager, or Employee).
authReference to the authentication manager.
zooReference to the main ZooGraph containing zoo data.

This static method acts as the main dispatcher. It determines the user’s privileges and routes execution flow to the correct interactive console menu.
