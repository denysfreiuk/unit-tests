Detailed Description

Provides authentication and account management functionality.

The AuthManager acts as an intermediary between the application logic and the account database. It provides a consistent interface for:

  • authenticating users,
  • adding new accounts,
  • editing and removing existing ones,
  • listing registered usernames.
Note
This class operates on top of the AccountRepository and depends on it for persistent data storage and retrieval.
