Detailed Description

Represents a weighted graph structure for the zoo system.

The Graph class models the layout of aviaries and paths connecting them. It supports common graph operations such as adding/removing vertices or edges, finding paths, and checking connectivity.

The graph uses an adjacency-based design, storing vertices in an unordered_map and edges in a vector. Paths can be computed using traversal or weighted search.

Note
This class serves as the base for ZooGraph, which adds zoo-specific logic.
