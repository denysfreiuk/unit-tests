Detailed Description

Centralized manager for handling animal operations.

The AnimalManager is responsible for creating, assigning, feeding, and relocating animals between aviaries. It maintains an in-memory map of all animals for efficient access and ensures synchronization with the underlying repository and zoo structure.

Note
This class is part of the ZooGraph subsystem and interacts with both ZooGraph and AnimalRepository.
