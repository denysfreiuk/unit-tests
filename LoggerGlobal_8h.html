This header exposes a single global Logger object that can be used across the entire application for consistent logging. It allows all modules to log messages without creating their own local instances.

The global logger simplifies integration between subsystems by providing a shared logging interface. It is typically defined in LoggerGlobal.cpp and initialized once at program startup.
