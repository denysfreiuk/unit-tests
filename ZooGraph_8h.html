Detailed Description

Declaration of the ZooGraph subsystem used in the Zoo Management System.

Author
Denys Freyuk
Date
27.10.2025
Version
1.1

This header defines the core classes representing the spatial graph of the zoo:

  • Aviary — a vertex representing an animal enclosure;
  • Path — a weighted connection between aviaries;
  • ZooGraph — the main graph structure combining all zoo entities and repositories.

The ZooGraph integrates database repositories, employee and animal managers, and graph traversal algorithms. It forms the core of the system’s spatial logic.
