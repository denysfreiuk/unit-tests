Detailed Description

Base abstract class for all animals in the zoo.

The Animal class encapsulates shared attributes (name, species, age, weight, type, aviary assignment, feeding status) and behaviors common to all animals. Derived classes must implement the makeSound() method to specify the sound characteristic for that animal group.

Each animal can belong to a specific aviary, has a feeding status, and may be interacted with through polymorphic calls. The class supports behavior simulation such as moving, feeding, and compatibility checks.
