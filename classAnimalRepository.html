Detailed Description

Provides database operations for Animal entities.

The AnimalRepository encapsulates low-level database logic for handling Animal objects. It allows insertion, removal, migration between aviaries, and full retrieval of all stored animals.

Note
This class is used by AnimalManager to synchronize in-memory data with the persistent database layer.
