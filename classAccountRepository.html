Detailed Description

Provides persistent storage and retrieval operations for Account objects.

The AccountRepository encapsulates all database operations related to user accounts. It enables the creation, modification, deletion, and querying of user credentials, maintaining a consistent interface for the authentication system.

Note
This class depends on DatabaseManager for executing SQL operations.
See also
