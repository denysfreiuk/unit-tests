Detailed Description

Centralized manager for handling all employee-related operations.

The EmployeeManager maintains an internal map of all zoo employees, allowing efficient querying, assignment management, and synchronization with persistent storage. It ensures that employees are properly assigned to aviaries and can be reassigned or removed when necessary.

Note
This class interacts closely with ZooGraph and EmployeeRepository.
